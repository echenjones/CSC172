The code for encode and decode is run in the main of HuffmanSubmit with the Alice in Wonderland text file and the Rush Rhees jpg. The implementation of the Huffman tree is from the textbook. I’ve created a method printHuffTree that prints out a rough visual representation of a Huffman tree with spaces before a node representing the number of levels down it is. As a note, it’s easier to read when it’s with a small number of characters.
